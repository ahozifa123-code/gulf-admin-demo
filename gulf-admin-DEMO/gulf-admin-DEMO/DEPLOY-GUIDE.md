# 🚀 دليل رفع الديمو على الإنترنت مجاناً
## Railway.app — أسهل طريقة (5 دقائق)

---

## الطريقة الأولى: Railway (الأسرع ✅)

### الخطوات:

**1. أنشئ حساباً على GitHub**
- اذهب لـ [github.com](https://github.com) وأنشئ حساباً مجانياً

**2. ارفع الملفات على GitHub**
- اذهب لـ [github.com/new](https://github.com/new)
- اسم المستودع: `gulf-admin-demo`
- اختر **Private** (خاص)
- اضغط **Create repository**
- ارفع جميع ملفات المجلد (اسحب وأفلت أو استخدم Upload files)

**3. افتح Railway**
- اذهب لـ [railway.app](https://railway.app)
- سجّل دخول بحساب GitHub
- اضغط **New Project**
- اختر **Deploy from GitHub repo**
- اختر مستودع `gulf-admin-demo`
- اضغط **Deploy Now**

**4. احصل على رابطك**
- بعد 2-3 دقائق، اضغط على **Settings** ← **Domains**
- اضغط **Generate Domain**
- ستحصل على رابط مثل: `gulf-admin-demo.railway.app`

**5. أرسل الرابط للعملاء!** 🎉

---

## الطريقة الثانية: Render.com (بديل مجاني)

1. اذهب لـ [render.com](https://render.com) وأنشئ حساباً
2. اضغط **New** ← **Web Service**
3. اربط مستودع GitHub
4. الإعدادات:
   - **Build Command:** `npm install`
   - **Start Command:** `node server.js`
   - **Environment:** Node
5. اضغط **Create Web Service**
6. انتظر 3-5 دقائق → ستحصل على رابط `.onrender.com`

> ⚠️ ملاحظة Render: الخدمة المجانية تنام بعد 15 دقيقة من عدم الاستخدام
> (أول طلب يأخذ 30 ثانية للاستيقاظ) — استخدم Railway لتجنب ذلك.
